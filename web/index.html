Servlet - Converte Celsius para Fahrenheit

Para testar:
- Via GET: Digitar http://localhost:8080/Celsius2Fahrenheit/Converte?Celsius={Valor}, onde {Valor} é a temperatura em graus celsius.
- Via POST: Definir host = http://localhost:8080/Celsius2Fahrenheit/ e submeter no corpo da mensagem o parâmetro "Celsius" com a temperatura em graus celsius. Usar add-on Postman do Google Chrome.
